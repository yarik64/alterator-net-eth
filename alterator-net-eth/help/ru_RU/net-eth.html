Ethernet-интерфейсы

Ethernet —обычный тип интерфейса для подключения к локальной сети.

IP-адрес — обязательный параметр узла IP-сети, который идентифицирует его уникальным для данной локальной или глобальной сети.

Маска сети — общий параметр IP-сети, который должен быть одинаковым для всех её хостов. От её значения зависит, к каким хостам локальной сети эта машина будет "знать" прямой маршрут.

Шлюз по умолчанию — хост в достижимой сети, который будет использоваться при неизвестности прямого маршрута до какого-либо IP-адреса.

В случае доступности DHCP-сервера необходимые параметры возможно получить автоматически — просто включите "Использовать DHCP".

Если в компьютере имеется несколько сетевых карт, то возможна ситуация, когда при очередной загрузке ядро присвоит имена интерфейсов (eth0, eth1) в другом порядке; нормально работать они так не будут. Чтобы этого не происходило, можно привязать интерфейс к имени по его аппаратному адресу (MAC) или местоположению на системной шине.
